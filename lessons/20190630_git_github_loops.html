    1. Loops
      1. for - usually loops a number of times or over a number of things
      2. while - loops until a test is false
        1. watch for unintended infinite loop

      5. continue - immedaitely go to top of loop

    2. % (modulus, mod) operator

      1. for printing changes that must vary based on the value of variables
      2. {} place holder used inside the string
      3. can put a "tag" inside the braces - then the variables used must be referenced by the tag, e.g. format('Max hp is: {max}'.format(max=maxhp)
